# Image Gallery Drag n' Drop

## Description

This is a simple image gallery that allows you to drag and drop images to reorder them. User have to create an account to use the app. The app uses Firebase for authentication and storage.

## Installation

1. Clone the repository
2. Run `npm install`
3. Run `npm start`

## Usage

1. Create an account
2. Drag and drop images to reorder them

## Default User

- Email: user@example.com
- Password: 1Password
